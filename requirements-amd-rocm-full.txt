# WHY THIS FILE EXISTS

#   compose. This file expands it with the base runtime deps for operators who
#   want a standalone host venv instead of the validated Docker image.
#

#   requirements.txt        ->  this file (ROCm)         reason

#   (no torchaudio)         ->  torchaudio ROCm 7.2.4     match torch build
#   (no triton-rocm)        ->  triton ROCm 7.2.4         match torch build
#   numpy==2.1.2            ->  numpy==2.1.2              base/CI train parity
#   scikit-learn==1.5.2    ->  scikit-learn==1.5.2      base/CI train parity
#   lightgbm==4.5.0        ->  lightgbm==4.5.0          base/CI train parity
# All other pins are runtime pins from requirements-base.txt.
#
# XGBoost upstream GPU support is CUDA-only; no official AMD ROCm XGBoost wheel is
# pinned here. The XGBoost pin stays CPU-safe.
#
# Build:  python3.12 -m venv .venv-rocm
#         .venv-rocm/bin/pip install --require-hashes -r requirements-amd-rocm-full.txt
# Prove:  .venv-rocm/bin/python tools/validate_rocm_acceleration.py --require-gpu
